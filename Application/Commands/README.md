# Commands

Этот подкаталог содержит классы команд, которые представляют собой запросы на выполнение определенных действий в приложении. Каждая команда инкапсулирует данные, необходимые для выполнения операции, и может содержать информацию о том, что именно должно быть сделано (например, создание, обновление или удаление сущности).
